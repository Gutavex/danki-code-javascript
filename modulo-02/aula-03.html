Trabalhando com Arrays

Arryas possuem características específicas de lista

Arryas também possuem seus métodos específicos para manipulá-los

Site: MDN: Array Definition

tabela de exemplos de arrays

  • array - lista de valores
  • entries - posição do valor no array
  • every - testa os elementos do array
  • push - adicionar valor no array
  • pop - remover valor no array
  • length - tamanho do array
  • reverse - inverter a ordem do array
  • join - juntar valores do array
  • *split - separar valores do array - é um método string e não do array
  • splice - remover valores do array
  • slice - copiar valores do array
  • filter - cria um novo array com todos os elementos que passaram no teste
  • concat - concatenar arrays
