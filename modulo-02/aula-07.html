Trabalhando com Laços de Repetição

Exemplos de laços de repetição

  • for : é como se você tivesse uma lista de crianças e soubesse exatamente quantas são.
  • while : é como se você estivesse distribuindo balões enquanto ainda há crianças na fila.
  • do while : é como se você tivesse que entregar pelo menos um balão, mesmo que não tenha certeza se há crianças na fila.
  • for in : é como se você estivesse verificando as características de cada criança na festa. Você olha para cada característica (nome, idade, altura) e faz algo com essa informação.
  • for of : é como se você tivesse uma cesta de balões de diferentes cores e quisesse entregar um balão de cada cor para as crianças. Você pega um balão de cada vez e entrega.
